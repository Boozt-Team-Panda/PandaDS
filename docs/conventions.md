# Conventions

Naming rules, decision process, and governance for Panda DS.

*Populated from panda-core.md as decisions become stable.*

## Decision process

- Design sessions (Facu + Justin): decisions enter registry with `status: draft`
- Sync sessions (with devs): draft decisions are ratified → `status: stable`
- Dev input can modify a draft: old entry gets `supersededBy`, new DS-NNN is created
- Nothing is ever deleted — only superseded
